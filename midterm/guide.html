Style Guide

For this project I wanted to work on a website that would be interesting, educational, and playful. Accordingly, I decided to design a narrative and interactive website in which the user would be able to rewrite the fairy tales of our childhood with multiple choice questions.
For the style of the website, I decided to work with fonts and color palettes that reminds us of fairy tales, and merge them with more modern images, backgrounds, and content. I believe that it would efficiently reflect the message that I want to convey through this website.

Website Form

For this website, I want the user to follow the stories on a digital map of a modern city such as the ones we usually use on our phones or laptops to go somewhere. The goal is for the person to visualize the paths of each character, and how this path changes depending on the responses given to the multiple choices questions.
